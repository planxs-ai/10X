프롬프트, 3개의 레벨

레벨구조결과 품질예시
Lv.1 기본역할 + 형식 + 맥락60점"보고서 써줘"
Lv.2 체인단계별 분해 + 중간검증80점"먼저 목차 → 검토 → 본문"
Lv.3 전문가판단기준 + 비교분석 + 자기검증95점"3안 비교 → 최적안 → 근거"

오늘 목표: Lv.2 + Lv.3를 실전으로 체득

Lv.1 vs Lv.2 — 결과가 이렇게 다릅니다

Lv.1 기본

"내년도 예산 요구서 작성해줘"

→ 일반적인 내용, 우리 부서 맥락 없음
→ 수정에 또 시간 소요

→
Lv.2 체인

"먼저 예산 요구서에 필요한 항목 목차를 정리해줘"
→ (결과 확인 후)
"이 목차에 맞춰 본문을 작성해줘"

→ 구조가 잡힌 상태에서 작성
→ 수정 최소화

핵심: 한 번에 다 시키지 말고, 단계를 나눠라

Lv.3 전문가 — AI에게 "생각의 틀"을 준다

★ 오늘 강의의 킬러 파트

현실

공무원의 고민: "HWP의 벽"

"AI 좋은 건 아는데…
우리는 HWP 쓰잖아요"

AI(Claude, ChatGPT)는 HWP 파일을 직접 인식하지 못합니다

하지만, PDF 변환 30초면 해결됩니다

HWP → AI 연동, 이렇게 합니다

방법소요 시간서식 보존추천
① HWP → PDF 변환 → AI 업로드 30초 90%+ ⭐ 추천
② HWP 텍스트 복사 → AI 채팅창 붙여넣기 10초 텍스트만 빠른 전달용

PDF 변환 한 번이면, AI가 공무원 문서를 읽고 쓸 수 있습니다

6개월 수익 포트폴리오 프롬프트

탑다운 분석 프롬프트 (1/2)

"당신은 20년 경력의 한국 주식시장 수석 애널리스트입니다.
다음 조건으로 6개월 내 수익 확률이 가장 높은 포트폴리오를 설계하세요.

[Step 1] 매크로 환경 분석
현재 시점(2026년 4월) 기준으로:
- 한국 기준금리 방향 (인상/동결/인하)
- 원/달러 환율 추세 + 외국인 자금 흐름
- 글로벌 리스크 요인 (지정학, 유가, 미중 관계)
- WGBI 편입 효과 + 반도체 슈퍼사이클 현황
위 4가지를 분석하고, 지금 돈이 흘러가는 방향을 판단하세요.

[Step 2] 유망 섹터 3개 선정
매크로 분석 결과를 기반으로,
향후 6개월 수익 확률이 높은 한국 섹터 3개를 선정하세요.
각 섹터의 선정 근거를 (정책 수혜 / 실적 성장 / 수급 흐름) 기준으로 설명."

킬러 프롬프트 — 하단

6개월 수익 포트폴리오 프롬프트 (계속)

탑다운 분석 프롬프트 (2/2)

[Step 3] 주도주 선정 — 각 섹터 2종목
각 섹터에서 아래 채점 기준으로 상위 2종목, 총 6종목을 선정하세요.

[채점 기준 — 100점 만점]
① 실적 모멘텀 (영업이익 증가율, 컨센서스 상회 여부) — 30점
② 수급 (외국인+기관 순매수 추세 최근 1개월) — 25점
③ 밸류에이션 (PER·PBR 동종 대비 저평가 정도) — 25점
④ 촉매 (신사업, 정책 수혜, 수주 등 6개월 내 이벤트) — 20점

[Step 4] 자기검증 + 포트폴리오 구성
- 6종목 간 상관관계가 높지 않은지 검증
- 섹터 편중 리스크, 최근 1개월 악재 뉴스 점검
- 문제 발견 시 대체 종목 교체
- 최종 6종목의 투자금 1,000만원 비중 배분
- 종목별 매수 희망가, 손절가(-7%), 목표가(+15%~20%) 제시
- 6개월 예상 수익률 시나리오 (최악/보통/최선)"
